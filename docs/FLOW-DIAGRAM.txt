Result:
-------
✓ Immutable artifacts (JARs, POMs) → S3 (persistent, shared across pods)
✓ Mutable metadata (tracking files) → EmptyDir (ephemeral, per-pod)
✓ No random I/O on S3 mount
✓ Maven resolver works correctly with full metadata support


Configuration:
--------------
MAVEN_OPTS:
  -Dmaven.repo.local=/maven/.m2/repository-metadata
  -Ds3.resolver.artifactDir=/home/maven/.m2/repository

Volume Mounts:
  /home/maven/.m2/repository      → PVC (S3 via Mountpoint)
  /maven/.m2/repository-metadata  → EmptyDir
